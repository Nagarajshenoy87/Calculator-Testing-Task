#Simple Calculator Application – Test Cases

#Project Overview
This project contains detailed manual test cases designed for a Simple Calculator Application that performs:

+ Addition
- Subtraction
* Multiplication
/ Division

The test cases cover both **valid scenarios** and **invalid scenarios** including edge cases.

---

Objective
To verify the correctness, reliability, and error handling of the calculator application using structured manual testing techniques.

Test Coverage
✅Valid Input Testing
- Positive numbers
- Negative numbers
- Decimal numbers
- Multiple operations (BODMAS rule)

❌Invalid Input Testing
- Division by zero
- Non-numeric input
- Invalid characters

Test Case Structure
Each test case includes:

- Test Case ID
- Test Description
- Preconditions
- Test Steps
- Expected Result
